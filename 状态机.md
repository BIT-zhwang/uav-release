# 航点与状态机使用说明

## 状态机
 状态机与航点相关的程序使用python语言进行编写，利用rospy与ros节点进行数据交换；  
 一个简单的状态机程序在smatch.py程序中，通过修改main函数修改飞机的航线。

 ```
 cd elf
 python smatch.py
 ```

## px4飞行指令

* ### 起飞
```
rosrun px4code takeoff
```

* ### 降落
```
rosrun px4code land
```
